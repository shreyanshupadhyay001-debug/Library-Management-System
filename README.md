1. Project Topic

Library Book Management System (Simple Python-Based Project)

2. Short Problem Statement

Managing books manually in a small library often leads to errors, difficulty in tracking book availability, and time-consuming entries.
There is no simple automated system to add books, issue or return books, or view which books are available.

To address this, a basic Python-based library management system is proposed.

3. Objectives

To develop a simple system that can add, store, and manage book information in a structured way.

To automate the issuing and returning of books.

To provide quick access to available book records with minimal manual work.

4. Functional Requirements

The system must allow adding new books with title, author, and unique ID.

The system shall store book information in a lightweight format (JSON).

The system must allow issuing a book and updating its status.

The system must allow returning a book and restoring its availability.

The system must display all available books upon user request.

The system should maintain updated data even after program closure.
